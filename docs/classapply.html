Detailed Description

A utility class for applying functions to different data samples.

Functions using apply are applied on the currently selected dataset, unless mentioned otherwise. The dataset name is recovered from config.GetSetting('dataset').

ToEach applies a function on each of the data samples in the dataset.

The target function should have arguments in the format of (hsIm, , ...), where hsIm is an instance of class 'hsi'. The targetID is saved for figure saving purposes in config::'fileName'.
