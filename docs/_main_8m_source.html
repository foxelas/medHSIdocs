28%> @section Description
29%>
30%> The MedHSI package allows you to read and process Hyper-Spectral Images (HSI)
31%> captured with a TopCon Spectroradiometer (.h5 files).
32%>
33%> This package contains two branches :
34%> - medHSIMat branch for MATLAB 2020
35%> - Used for data import, preprocessing, normalization, dimension reduction and graphics production
36%> - medHSIpy branch for python 3.8
37%> - Used classification and segmentation tasks
38%>
